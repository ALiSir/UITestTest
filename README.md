# APP内容

2.登录成功界面 </br>

# 测试内容
### 1.测试登录失败情景 </br>
  `打开APP -> 点击登录 -> 登录失败`
### 2.测试登录成功情景 </br>
  `输入用户名 -> 输入密码 -> 登录成功 -> 跳转到登录成功界面`
